WordPress Plugin

Photocard Generator

এটি একটি WordPress Plugin। এই ফাইলগুলো আপনার WordPress সাইটের /wp-content/plugins/ ফোল্ডারে আপলোড করুন।

📦 ইনস্টলেশন

  1. wordpress-photocard ফোল্ডারটি /wp-content/plugins/ এ আপলোড করুন
  2. WordPress Admin → Plugins থেকে Photocard Generator activate করুন
  3. Settings → Photocard Generator থেকে কনফিগার করুন
