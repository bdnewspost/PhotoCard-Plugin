# Photocard Generator - WordPress Plugin

পোস্টের থাম্বনেইল থেকে ফটোকার্ড তৈরি ও ডাউনলোড করার WordPress প্লাগইন।

## Features

- পোস্ট থাম্বনেইল থেকে ফটোকার্ড জেনারেট
- একাধিক টেমপ্লেট (Classic, Modern, Elegant, News24)
- টাইটেল এডিটর (লাইন-ওয়াইজ কালার সাপোর্ট)
- বাংলা/ইংরেজি ভাষা সাপোর্ট
- কাস্টম ফন্ট, কালার, লোগো
- ডাউনলোড পারমিশন কন্ট্রোল
- সোশ্যাল মিডিয়া লিঙ্ক
- বিজ্ঞাপন স্লট সাপোর্ট

## Installation

1. `wordpress-photocard` ফোল্ডারটি `/wp-content/plugins/` এ আপলোড করুন
2. WordPress Admin → Plugins থেকে activate করুন
3. Settings → Photocard Downloader থেকে কনফিগার করুন
