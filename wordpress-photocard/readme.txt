== Description ==

Photocard Generator creates stunning 1080x1080 social media cards from your WordPress posts. Features include:

* 7 built-in templates: News24, Kalbela, Prothom Alo, Daily Star, Jugantor, Samakal, Daily Shadhin
* Full featured image controls: object-fit, position, zoom, per-side padding/border/radius
* Card background: solid color or gradient with direction control
* Template-specific color customization
* Social media icons with configurable font size (Facebook, Instagram, YouTube, LinkedIn, Website)
* Title editor with word-wise color, bold, italic formatting
* 25+ Bengali & English font support
* Date display with position control and Bengali/Hindi/English formatting
* Logo with position control (left/center/right)
* Custom background overlay image support
* Card border with width, color, radius, and image border support
* Download permission control (Everyone/Logged in/Author/Editor/Admin)
* Share to Facebook, Instagram, WhatsApp, Twitter, LinkedIn
* High-quality PNG export (2x to 5x)
* Admin post column with quick download link
* License-based activation system
